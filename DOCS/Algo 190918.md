### 비트연산

- 비트 연산자

```
&: AND
|: OR
^: XOR (같으면 0 다르면 1)
~: 모든 비트 반전
<<: 비트 열을 왼쪽으로 이동
>>: 비트 열을 오른쪽으로 이동 
```

- 연산

```python
1 << n
"""
- 2**n개의 값을 갖는다.
- 원소가 n개인 경우의 모든 부분집합의 갯수
- Power Set (모든 부분 집합)
"""

i & (1 << j)
"""
- 계산 결과는 i의 j번째 비트가 1인지 아닌지를 의미 
"""
```
